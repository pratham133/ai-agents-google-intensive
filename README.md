# 📖 About

This repository documents my journey through Google's **AI Agents Intensive Program**.

The goal of this program is to gain hands-on experience with modern AI agent systems, agentic workflows, interoperability standards, AI-assisted software development, and production deployment practices.

Throughout the program, I am building projects, documenting learnings, completing hands-on labs, and creating a public portfolio of my progress.
